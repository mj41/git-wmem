# git-wmem Tools Use Cases

## UC: Basic Development Workflow
- 1) User starts [UC: git-wmem-init basic](use-cases/git-wmem-init/basic.md) (initializes `wmem-repo` including the first commit)
- 2) User runs [UC: user-sh-cmds wds-setup-basic](use-cases/user-sh-cmds/wds-setup-basic.md)
- 3) User runs [UC: git-wmem-commit basic](use-cases/git-wmem-commit/basic.md) (second commit in `wmem-repo`)
- 4) User runs [UC: user-sh-cmds wds-file-changes](use-cases/user-sh-cmds/wds-file-changes.md)
- 5) User runs [UC: git-wmem-commit basic](use-cases/git-wmem-commit/basic.md) (third commit in `wmem-repo`)
- 6) User runs [UC: user-sh-cmds wds-git-cmds](use-cases/user-sh-cmds/wds-git-cmds.md)
- 7) User runs [UC: user-sh-cmds commit-msg-prefix-file](use-cases/user-sh-cmds/commit-msg-prefix-file.md)
- 8) User runs [UC: git-wmem-commit basic](use-cases/git-wmem-commit/basic.md) (fourth commit in `wmem-repo`)
- 9) User runs [UC: git-wmem-log basic](use-cases/git-wmem-log/basic.md)
