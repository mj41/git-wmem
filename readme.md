# git-wmem

Tools to track uncommitted changes in multiple git repositories.

## Documentation

See `docs/` directory:
- [use-cases](docs/use-cases.md) - use cases for `git-wmem` tools
- [data-structures](docs/data-structures.md) - data structures used in the `git-wmem` tools
- [dictionary](docs/dictionary.md) - terms used in the documentation
- [boundaries](docs/boundaries.md) - boundaries of `git-wmem` tools
- [validations](docs/validations.md) - validations for the `git-wmem` tools

## End-to-End Tests

The `e2e/` directory contains comprehensive Go-based tests that validate the complete functionality of git-wmem tools.

See [`tests_e2e/readme-e2e.md`](tests_e2e/readme-e2e.md) for detailed test documentation.
